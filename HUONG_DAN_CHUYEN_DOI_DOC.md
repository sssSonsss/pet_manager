# Hướng Dẫn Chuyển Đổi Sang File .doc

## Cách 1: Sử dụng Microsoft Word (Khuyến nghị)

1. **Mở file HTML bằng Word:**
   - Mở Microsoft Word
   - File → Open → Chọn file `BAI_TAP_LON.html`
   - Word sẽ tự động chuyển đổi HTML sang định dạng Word

2. **Lưu thành file .doc:**
   - File → Save As
   - Chọn định dạng: **Word 97-2004 Document (.doc)** hoặc **Word Document (.docx)**
   - Đặt tên file: `BAI_TAP_LON.doc`
   - Click Save

## Cách 2: Sử dụng Google Docs (Nếu không có Word)

1. **Upload file HTML lên Google Drive:**
   - Truy cập https://drive.google.com
   - Upload file `BAI_TAP_LON.html`

2. **Mở bằng Google Docs:**
   - Click đúp vào file HTML
   - Google Docs sẽ tự động mở và chuyển đổi

3. **Tải xuống dạng .doc:**
   - File → Download → Microsoft Word (.docx)
   - Hoặc File → Download → Word 97-2004 (.doc)

## Cách 3: Sử dụng LibreOffice (Miễn phí)

1. **Cài đặt LibreOffice** (nếu chưa có):
   - Tải từ: https://www.libreoffice.org/download/
   - Cài đặt trên máy

2. **Mở file HTML:**
   - Mở LibreOffice Writer
   - File → Open → Chọn file `BAI_TAP_LON.html`

3. **Lưu thành .doc:**
   - File → Save As
   - Chọn định dạng: **Microsoft Word 97-2003 (.doc)**
   - Click Save

## Cách 4: Sử dụng Online Converter

1. Truy cập các trang web chuyển đổi:
   - https://www.zamzar.com/convert/html-to-doc/
   - https://convertio.co/html-doc/
   - https://www.online-convert.com/

2. Upload file `BAI_TAP_LON.html`

3. Chọn định dạng đầu ra: .doc

4. Download file đã chuyển đổi

## Lưu ý

- File HTML đã được format sẵn với:
  - Font: Times New Roman, 12pt
  - Căn lề: 2.5cm trên/dưới, 2cm trái/phải
  - Định dạng heading, paragraph, list đã được thiết lập
  - Code blocks được format riêng

- Sau khi mở bằng Word, bạn có thể:
  - Điều chỉnh định dạng nếu cần
  - Thêm số trang
  - Thêm header/footer
  - Chèn hình ảnh (nếu có)

## File đã tạo

- ✅ `BAI_TAP_LON.html` - File HTML có thể mở bằng Word
- ✅ `convert_to_doc.py` - Script Python để tạo lại file HTML nếu cần
